# Frontend - Developer Directory

React frontend for the Developer Directory application with Tailwind CSS.

## Features

- React functional components with Hooks
- Form with validation
- Search and filter functionality
- Toast notifications
- Responsive design with Tailwind CSS
- Clean modern UI

## Setup

```bash
npm install
npm start
```

App runs on `http://localhost:3000`

## Components

- **App.js** - Main application component
- **DeveloperForm.js** - Form for adding developers
- **DeveloperList.js** - Display list of developers
- **SearchFilter.js** - Search and filter interface

## Tech Stack

- React 18.2.0
- Tailwind CSS 3.3.5
- Axios 1.6.0
- React Hot Toast 2.4.1

## Environment Variables

Create a `.env` file:

```
REACT_APP_API_URL=http://localhost:5000
```
